Basewind

A supercharged Tailwind config with simple, custom composition utilities inspired by Every Layout, plus fluid typography and spacing scales created with Utopia. Includes sensible global styles I find myself adding to nearly every project, along with a growing collection of general-purpose utilities.

Git Repo

Composition Utilities

Cluster

A layout that lets you distribute items with consitent spacing, regardless of their size.

Use .gap-* to control the gap between each cluster item.

Use any flexbox aligmnent value to configure how the items align. The defualt is .items-center

Repel

A little layout that pushes items away from each other where there is space in the viewport and stacks on small viewports.

Use any flexbox aligmnent value to configure how the items align. The defualt is .items-center

Stack

Use .space-y-* for simplicity.

Switcher

A layout that allows you to lay 4 items next to each other until there is not enough horizontal space to allow that.

Use .switcher-width-* to control how large the container needs to be to allow items to sit inline with each other. This accepts any value based on the container widths set in the theme.

Frame

A layout that allows you to control the aspect ratio of the element within it. Frame is mostly useful for cropping media (videos and images) to a desired aspect ratio. This accepts any value based on the aspect ratios set in the theme.

Stretch Link

The .stretch-link class is used to expand a link to cover the full parent container. This is to avoid the evil sin of wrapping the entire content in an anchor tag, which affects accessibility.

Make sure to add .relative to the parent element to prevent the link from stretching outside the parent element.

The only issue with this solution is that it stops the text from being highlighted.
